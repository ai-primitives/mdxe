# Basic MDX Example

This is a simple MDX file that demonstrates basic markdown features.

## Features

- **Bold text** and *italic text*
- Lists and nested lists
  - Nested item 1
  - Nested item 2
- [Links](https://mdxjs.com)
- Code blocks

```javascript
console.log('Hello, MDX!')
```

## Components

You can use components directly in your MDX:

<Button>Click me!</Button>
