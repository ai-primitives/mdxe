# {frontmatter.title}

This example demonstrates frontmatter usage for App Router metadata.

## Metadata Usage

The frontmatter above exports:
- title and description for SEO
- keywords for meta tags
- custom viewport configuration
- revalidation time

## Dynamic Content

You can use frontmatter values in your content:
- Title: {frontmatter.title}
- Description: {frontmatter.description}
